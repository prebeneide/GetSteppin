# Steppin - Skrittteller App

En sosial skrittteller app for iOS og Android hvor brukere kan:
- Registrere seg og legge til venner
- Sette daglige skrittmål
- Samle emojis gjennom ulike prestasjoner
- Se grafer og sammenligne seg med venner
- Delta på topplister

## Teknologi

- React Native (iOS og Android)
- Supabase (Backend, Database, Authentication)
- TypeScript

## Funksjoner

### Brukerfunksjonalitet
- Brukerregistrering og innlogging
- Legge til venner etter brukernavn
- Sette daglige skrittmål

### Emoji-achievements
- 🍒 Kirsebær - for hver 1000m
- 🍑 Fersken - for hver 10000m
- 🏆 Pokal - vinner over alle venner
- 👑 Ukesvinner - topp ukesvinner
- ⭐ Topp 5% - topp 5% av alle brukere
- ✅ Dagens mål - klart dagens mål

### Visualisering
- Grafer for daglige, ukentlige, månedlige og årlige data
- Sammenligning med venner
- Topplister (venner og alle brukere)

### Sosialt
- Se når venner er aktive
- Se venners emoji-samlinger
